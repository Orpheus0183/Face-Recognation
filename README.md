# Face Recognition
Merupakan sistem yang dibuat untuk membandingkan apakah dua foto orang itu merupakan orang yang sama atau tidak.

cara menggunakan:
1. masuk ke link yang tersedia
2. kalau ingin langsung bisa ke halaman 3 dan membandingkan 2 foto
3. kalau ingin hasil lebih akurat, bisa training data set dulu di halaman 1 atau upload file .pkl
